# Vlad-Search
A small search engine project using bing's search API

## Viewing
At the current stage, the project is not hosted.

#### Step 1. 
```bash
Clone the repository
```
#### Step 2. 
```bash
cd vlad-search
```
#### Step 3. 
```bash
npm install
```
#### Step 4. 
```bash
type "quasar dev" in the terminal
```
